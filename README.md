# rv32imac_mmu
A variant of Charles Lohr's mini-rv32ima that has a Sv32 MMU and C instruction support in many programming languages.

# Why?

It's very cool and intentionally portable :3 

<sub><small>i was bored also</small></sub>

# Planned languages: basically all mainstream ones, but so far:

JavaScript: The interpreter version works, however im working on the RVJSCG JIT (RVJSCode Generator) compiler, similar to QEMU's TCG. 


# The JavaScript version is the base for every other version, JIT and all (but JS in RVJSCG changed for the language, like RVCCG or RVPYCG.).
